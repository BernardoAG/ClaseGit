Terminos importantes:

Element- son 'tags' utilizados para mostrar contenido del HTML, se escriben entre < > y terminan con un /.
< p> = párrafo
< br> = break line
< pre> = formato predefinido (bueno para poemas)

Attribute- es algo de un elemento que da más información de él. El formato suele ser: name="value"
P.e- < img src="img_girl.jpg" width="500" height="600" >

title= tooltip
heigh = altura
width= ancho
alt = texto si no sale imágen
href = link
style = cambiar tamano/ color/ tipo de letra

Headings-
< html> la raíz de un documento HTML
< body> cuerpo del documento
< head> contiene todos los elementos de cabeza (title, scripts, styles, etc.)
< h1> a < h6> titulos de HTML
< hr> línea para cambio de tema

Style-
text-align: POSICIÓN
font-family: TIPO_DE_LETRA
font-size: TAM
background: COL
< b> - Bold text
< strong> - Important text
< i> - Italic text
< em> - Emphasized text
< mark> - Marked text
< small> - Small text
< del> - Deleted text
< ins> - Inserted text
< sub> - Subscript text
< sup> - Superscript text
Quotation-
< abbr> abreviación
< address> información de contacto
< bdo> direccion de texto
< blockquote> quote de otra dirección
< cite> título de una obra
< q> quote corta

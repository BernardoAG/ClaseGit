Terminos importantes:

Element- son 'tags' utilizados para mostrar contenido del HTML, se escriben entre < > y terminan con un /.

< p> = párrafo
< br> = break line
< pre> = formato predefinido (bueno para poemas)

Attribute- es algo de un elemento que da más información de él. El formato suele ser: name="value"
P.e- < img src="img_girl.jpg" width="500" height="600" >

title= tooltip
heigh = altura
width= ancho
alt = texto si no sale imágen
href = link
style = cambiar tamano/ color/ tipo de letra

Entities

Uniform Resource Locator (URL)
Formato:
scheme://prefix.domain:port/path/filename
scheme - tipo de servicio de interner (comunmente http o https)
prefix - prefix del dominio (www)
domain - defines the Internet domain name (like w3schools.com)
path - el camino en el servidor
filename - el nombre del archivo

XHTML- eXtensible HyperText Markup Language: es una versión más estricta de HTML, en donde se deben respetar ciertas reglas
Form- es para hacer cuestionarios o recibir input del usuario.
Tipos:
text- una línea de texto
radio- botón para opciones múltiples
submit- para subir la información

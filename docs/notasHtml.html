Uniform Resource Locator (URL)

Formato:
  • scheme://prefix.domain:port/path/filename
  • scheme - tipo de servicio de interner (comunmente http o https)
  • prefix - prefix del dominio (www)
  • domain - defines the Internet domain name (like w3schools.com)
  • path - el camino en el servidor
  • filename - el nombre del archivo
XHTML- eXtensible HyperText Markup Language: es una versión más estricta de HTML, en donde se deben respetar ciertas reglas

Tablas:
Ejemplo
< table> define una tabla
< tr> una fila de la tabla
< td> datos de una tabla
< th> heading de una tabla
< caption> titulo de una tabla
nth-child(even/odd) sirve para dividir las filas en colores

Display- se puede usar display o visibility para mostrar elementos de diferentes modos
  • display: none
  • visibility: hidden

Max-height, max-width- se utiliza para que un elemento no rebace tal tamaño, se ajusta a que cambien de tamaño la ventana.

Overflow- se encarga de los casos en donde el texto no cabe dentro del tamaño de un elemento.
  • visible- ignora el tamaño y se sale del elemento
  • hidden- lo corta en donde el tamaño lo permite
  • scroll- agrega scrollbars para no perder el texto
  • auto- scroll solo si es necesario
  • Hay overflow-x y overflow-y, dependiendo de que eje quieres considerar

Clear / float- sirve para organizar imagenes o elementos para que 'floten' juntos
  • Al terminar debes de poner lo siguiente para regresar al formato normal:
    .clearfix::after {
    content: "";
    clear: both;
    display: table;     }

Combinators- sirve para seleccionar algunos elementos en específico.
  • descendant selector (space)- se refiere a los que están dentro del otro elemento
  • child selector (>)- se refiere a los que van justo un nivel adentro de otro elemento, "hijos"
  • adjacent sibling selector (+)- van justo despues de otro elemento
  • general sibling selector (~)- todos los hermanos del otro elemento

Pseudo-classes- sirven para darle formato especial a diferentes estados de un elemento (links por ejemplo)
  • Formato:
    selector:pseudo-class {
    property:value;
    }
  • First-child- puedes ustar esta pseudoclase para seleccionar solo la primera instancia de un elemento

Pseudo-elements- similares a la pseudoclase, pero este sirve para darle formato a ciertas partes de un elemento (por ejemplo la primer línea o letra)
  • ::after
  • ::before
  • ::first-letter
  • ::first-line
  • ::selection

Navigation bar- hecha a base de links
  • Vertical- se hace con una lista quitandole sus puntos
  • Horizontal- se hace con un div o con una lista con elementos 'inline'
  • Para hacerla que se quede todo el tiempo, le pones position:fixed, y le determinas un tamaño

Attribute selector- sirve para seleccionar solo elementos que contienen algun atributo
  • se escribe como: elemento[atributte] {}
  • puedes checar que el atributo tenga tal valor, usando [attribute= "value"]
